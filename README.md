# ShoppyGlobe API

## About
This is a simple RESTful API for managing products using Node.js and Express.

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start the server with `npm start`

## API Routes
- GET /products
- POST /products
- PUT /products/:id
- DELETE /products/:id
